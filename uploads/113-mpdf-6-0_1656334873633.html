This is a Joomla! library package for mPDF - Unicode-HTML Free PDF generator, version 6.0. CMExtension only created this package for installing into Joomla! CMS as a Joomla! library. All credits for mPDF belong to Ian Back and the contributors.

For more information about mPDF, please visit http://www.mpdf1.com.

Note: If you use Microsoft Windows, you could have problem in extracting this ZIP package. Instead of downloading the package from this page please download an alternative package here: https://github.com/cmextension/mpdf (click "Download ZIP" button on the right side).
